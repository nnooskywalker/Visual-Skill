# VisualSkill 🎨💻
VisualSkill is a modern online learning platform developed as part of the Human and
Computer Interaction for final project. The website is designed to provide and engaging and
responsive learning experience for users who want to explore creative and digital skill courses.

This project focuses on applying Human-Computer Interaction (HCI) principles through
intuitive navigation, consistent design, responsive layouts, and interactive user experiences.
The platform includes multiple pages, such as Home, Courses, Careers, About Us, and Register page with form
validation using JavaScript.
# ✨ Features
- Responsive website design for desktop, tablet, and mobile devices
- Interactive and modern UI/UX
- Landing page with promotional sections
- Course catalog with descriptions and images
- Career page with FAQ section
- About Us page containing the company profile and vision
- Registration form with JavaScript validation
- External CSS implementation
- Mobile-friendly navigation using media queries

# 📱 Responsive Design
The website is fully responsive and optimized for:
- Desktop
- Tablet
- Mobile Devices

Media queries are implemented for screens below 700px width to ensure accessibility and
usability across different devices.
# 🎯 Project Objective
The purpose of this project is to design and develop a user-friendly educational platform
while implementing the principles of Human and Computer Interaction (HCI), including:
- Consistency
- Accessibility
- Visual hierarchy
- User-centered design
- Responsive interaction
